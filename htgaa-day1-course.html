The Central Dogma of Biology

How information flows from genes to proteins

What is the Central Dogma?

The Central Dogma is the most fundamental concept in molecular biology. It describes how the genetic information stored in your DNA gets converted into functional proteins that do the actual work in your cells.

The Central Dogma: genetic information flows from DNA to RNA to Protein

Image: Wikimedia Commons (Public Domain)

üíª Software Analogy

If you're familiar with software development, think of it this way:

  • DNA = Source code (stored in a repository, rarely modified directly)
  • RNA = Compiled bytecode (temporary working copy)
  • Protein = Running program (does the actual work)

The Three Key Processes

1. Replication (DNA ‚Üí DNA)

Before a cell divides, it needs to copy all of its DNA so each daughter cell gets a complete set of instructions. This copying process is called replication.

DNA replication: the double helix unwinds and each strand serves as a template for a new complementary strand

Image: Wikimedia Commons (Madprime, CC BY-SA 3.0)

  • Happens before cell division
  • Creates an exact copy of the entire genome
  • Uses an enzyme called DNA polymerase
  • Error rate: about 1 mistake per billion letters copied

2. Transcription (DNA ‚Üí RNA)

Transcription is the process of creating an RNA copy of a specific gene. Unlike replication, which copies everything, transcription only copies the genes that are needed at that moment.

  • Only copies specific genes as needed
  • Uses an enzyme called RNA polymerase
  • Produces messenger RNA (mRNA) that carries the message
  • Happens in the nucleus of eukaryotic cells

3. Translation (RNA ‚Üí Protein)

Translation is where the magic happens - the RNA message is "translated" into a sequence of amino acids that fold into a functional protein.

  • Happens at ribosomes (molecular machines)
  • mRNA is read in groups of 3 letters called codons
  • Each codon specifies one amino acid
  • Amino acids are linked together to form proteins
üìä By the Numbers
  • Human genome: ~3 billion DNA letters
  • Number of genes: ~20,000
  • Possible codons: 64 (but only 20 amino acids + stop signals)
  • Average protein: ~300 amino acids long

Why This Matters for Biotech

Understanding the Central Dogma is essential because every biotechnology intervention works by manipulating one of these steps:

  • Gene therapy: Changing the DNA sequence
  • RNA vaccines: Delivering mRNA directly (like COVID vaccines)
  • Protein drugs: Delivering functional proteins (like insulin)
  • CRISPR: Editing the DNA to change what proteins are made

DNA Structure & Replication

The molecule that stores genetic information

What is DNA?

DNA (Deoxyribonucleic Acid) is a long molecule that contains all the genetic instructions used in the development, functioning, and reproduction of all known living organisms.

DNA Double Helix Structure
The twisted ladder shape of DNA with base pairs forming the rungs

Wikimedia Commons (Zephyris, CC BY-SA 3.0)

DNA Chemical Structure
Sugar-phosphate backbone with nitrogenous bases paired in the middle

Wikimedia Commons (Madeleine Price Ball, Public Domain)

The Double Helix Structure

DNA has a distinctive shape called a double helix - imagine a twisted ladder:

  • The sides of the ladder: Made of sugar (deoxyribose) and phosphate groups
  • The rungs: Made of paired bases (A-T and G-C)
  • The twist: The whole structure spirals around an axis

The Four Bases

DNA is made up of four chemical bases, often abbreviated by their first letters:

Base Full Name Type Pairs With
A Adenine Purine (2 rings) T (2 hydrogen bonds)
T Thymine Pyrimidine (1 ring) A (2 hydrogen bonds)
G Guanine Purine (2 rings) C (3 hydrogen bonds)
C Cytosine Pyrimidine (1 ring) G (3 hydrogen bonds)

Why Base Pairing Matters

Complementary base pairing is crucial for:

  • Replication: Each strand can serve as a template to make the other
  • Transcription: RNA is made using one DNA strand as a template
  • DNA repair: If one strand is damaged, the other provides the correct sequence
  • CRISPR targeting: Guide RNAs use base pairing to find specific sequences

5' to 3' Directionality

DNA strands have a direction, determined by the chemical structure:

  • Each strand has a 5' end (pronounced "five prime") and a 3' end
  • DNA is always read and synthesized from 5' to 3'
  • The two strands run in opposite directions (antiparallel)
üíª Software Analogy

Think of 5' to 3' like reading direction. Just as English text is always read left-to-right, DNA is always read 5' to 3'. The molecular machinery that reads DNA can only work in this direction.

Gene Structure & Expression

How genes are organized and controlled

What is a Gene?

A gene is a segment of DNA that contains the instructions for making one specific protein (or sometimes RNA). Think of genes as individual recipes in a cookbook - the whole genome is the cookbook, and each gene is one recipe.

Parts of a Gene

A typical gene has several important regions:

Promoter
5' UTR
Coding Sequence (CDS)
3' UTR
Terminator

Structure of a typical gene (simplified)

Promoter

The promoter is the "ON switch" for the gene. It's a DNA sequence located before the gene that:

  • Is recognized by RNA polymerase (the enzyme that makes RNA)
  • Determines when and how strongly a gene is expressed
  • Can respond to signals (e.g., "turn on this gene when glucose is low")
üíª Software Analogy

Promoters are like function decorators or triggers. A strong promoter is like a function that gets called frequently; a weak promoter is called rarely. An inducible promoter is like a function triggered only by specific events.

5' UTR and 3' UTR

UTR stands for "Untranslated Region" - these regions are transcribed into RNA but don't code for protein:

  • 5' UTR: Controls how efficiently the mRNA is translated
  • 3' UTR: Affects mRNA stability and localization

Coding Sequence (CDS)

The Coding Sequence is the actual "recipe" - the part that gets translated into protein:

  • Starts with a start codon (ATG, which codes for methionine)
  • Ends with a stop codon (TAA, TAG, or TGA)
  • Everything in between codes for amino acids

Terminator

The terminator signals the end of transcription - it tells RNA polymerase to stop and release the RNA.

The Genetic Code

The genetic code is the "dictionary" that translates RNA sequences into amino acids:

  • Each group of 3 nucleotides is called a codon
  • There are 64 possible codons (4¬≥ = 64)
  • These code for 20 amino acids plus stop signals
  • Multiple codons can code for the same amino acid (redundancy)
üìä The Universal Code

The genetic code is nearly universal across all life on Earth - from bacteria to humans. This shared code is strong evidence that all life evolved from a common ancestor, and it's why we can put human genes into bacteria and have them work!

Gene Regulation

Not all genes are active all the time. Gene regulation controls which genes are expressed:

Activators

Proteins that increase gene expression by helping RNA polymerase bind or work more efficiently.

Repressors

Proteins that decrease gene expression by blocking transcription.

üíª Software Analogy

Gene regulation is like access control and feature flags:

  • Activators = Feature flags set to "enabled"
  • Repressors = Feature flags set to "disabled"
  • Inducers = User actions that toggle features on

‚úÖ Key Takeaways

  • Genes have distinct regions: promoter, UTRs, coding sequence, terminator
  • Promoters control when and how much a gene is expressed
  • The genetic code uses 3-letter codons to specify amino acids
  • Gene expression is regulated by activators and repressors

CRISPR & Genome Engineering

How we edit DNA with precision

What is CRISPR?

CRISPR (Clustered Regularly Interspaced Short Palindromic Repeats) is a revolutionary gene editing technology that works like "molecular scissors." It allows scientists to cut DNA at specific locations and make precise changes.

CRISPR-Cas9: The guide RNA (orange) directs Cas9 protein to the target DNA sequence, where it makes a double-strand break

Image: Wikimedia Commons (Marius Walter, CC BY-SA 4.0)

üèÜ Historical Note

CRISPR was discovered as a bacterial immune system. Bacteria use it to remember and destroy viruses! Jennifer Doudna and Emmanuelle Charpentier won the 2020 Nobel Prize in Chemistry for developing CRISPR-Cas9 as a genome editing tool.

How CRISPR-Cas9 Works

CRISPR-Cas9 has two main components:

1. Guide RNA (gRNA)

A short RNA molecule (~20 nucleotides) that:

  • Is designed by scientists to match the target DNA sequence
  • Uses complementary base pairing to find the target
  • Acts as a "GPS" to guide Cas9 to the right location

2. Cas9 Protein

The "scissors" enzyme that:

  • Binds to the guide RNA
  • Scans DNA looking for a matching sequence
  • Cuts both strands of DNA at the target site
  • Requires a PAM sequence (usually NGG) next to the target

What Happens After the Cut?

After Cas9 cuts the DNA, the cell tries to repair it. This can lead to different outcomes:

Two repair pathways after CRISPR cuts: NHEJ (error-prone, causes disruptions) vs HDR (precise editing with template)

Image: Wikimedia Commons (Viktoria.science, CC BY-SA 4.0)

Non-Homologous End Joining (NHEJ)

  • The cell's default repair mechanism
  • Often introduces small insertions or deletions ("indels")
  • Usually disrupts the gene (gene knockout)
  • Quick but error-prone

Homology-Directed Repair (HDR)

  • Uses a provided template DNA for precise repair
  • Allows precise insertions or corrections (gene knock-in)
  • Slower and less efficient
  • Requires providing a DNA template
üíª Software Analogy

NHEJ is like a quick patch that might break the code (useful if you want to disable a function).

HDR is like a careful refactor using a well-tested PR - more precise but requires more setup.

Phage Therapy

Viruses as weapons against bacteria

What Are Bacteriophages?

Bacteriophages (or simply "phages") are viruses that specifically infect and kill bacteria. The name literally means "bacteria eater" (Greek: phagein = to eat).

Bacteriophages Under Microscope
Electron microscope image showing phages with their characteristic head and tail structure

Wikimedia Commons (Dr. Graham Beards, CC BY-SA 3.0)

Phage Therapy Mechanism
Phages specifically target and destroy bacterial cells while leaving human cells unharmed

Wikimedia Commons (Guido4, CC BY-SA 4.0)

üìä Phages by the Numbers
  • 10¬≥¬π phages on Earth - the most abundant biological entities
  • Kill 40% of ocean bacteria every day
  • Discovered over 100 years ago (1915)
  • Each phage targets only specific bacterial strains

Why Phage Therapy Matters

We're facing an antibiotic resistance crisis:

  • 1.2+ million deaths per year from antibiotic-resistant infections
  • Very few new antibiotics in development
  • Bacteria evolve resistance faster than we can develop new drugs
  • Some infections are becoming untreatable

How Phages Work

The lytic cycle is how phages kill bacteria:

The lytic cycle: phage attaches, injects DNA, replicates inside the cell, assembles new particles, and bursts the cell

Image: Wikimedia Commons (Suly12, CC BY-SA 4.0)

  1. Attachment: Phage binds to specific receptors on bacterial surface
  2. Injection: Phage injects its DNA into the bacterium
  3. Replication: Bacterial machinery makes copies of phage DNA and proteins
  4. Assembly: New phage particles are assembled inside the cell
  5. Lysis: Cell bursts, releasing 50-200 new phages

Phages vs. Antibiotics

Feature Antibiotics Phages
Specificity Broad-spectrum (kill many bacteria) Highly specific (target one species/strain)
Effect on microbiome Disrupt beneficial bacteria Leave beneficial bacteria alone
Resistance Spreads between bacteria Phages co-evolve with bacteria
Self-amplifying No (need repeated doses) Yes (multiply at infection site)
Finding new ones Difficult, expensive R&D Found in environment (soil, sewage)
üî¨ Current State of Phage Therapy

Phage therapy is already used in some countries (Georgia, Poland, Russia) and is being explored in clinical trials worldwide. The FDA has approved compassionate use for patients with no other options. Major applications include treating wound infections, UTIs, and lung infections in cystic fibrosis patients.

Connection to Longevity

Phage therapy is particularly relevant for aging populations because:

  • Older individuals are more susceptible to infections
  • Antibiotic resistance is a major threat to elderly care
  • Phages could be used for microbiome engineering (selectively removing harmful bacteria)
  • Potential for treating chronic infections that affect healthspan

Genetic Circuits

Programming living cells

What Are Genetic Circuits?

Genetic circuits are engineered DNA systems that perform logical operations inside living cells. Just like electronic circuits process electrical signals, genetic circuits process biochemical signals.

A genetic toggle switch: two repressors that mutually inhibit each other, creating bistable (ON/OFF) behavior

Image: Wikimedia Commons (Vossman, CC BY-SA 4.0)

üíª The Ultimate Software Analogy

Genetic circuits are literally biological programming:

  • Promoters = Function calls (trigger expression)
  • Repressors = if (NOT x) statements
  • Activators = if (x) statements
  • Feedback loops = while loops
  • Ribosomes = The runtime environment

Circuit Components

Genetic circuits are built from standard biological parts:

Promoters

Control when a gene is expressed:

  • Constitutive: Always on (like a constant)
  • Inducible: Turned on by a signal (like an event listener)
  • Repressible: Turned off by a signal

Ribosome Binding Site (RBS)

Controls how much protein is made (like setting the volume).

Coding Sequence

The gene that produces the output protein (the function body).

Terminator

Signals the end of transcription (like a return statement).

Reporters

Proteins that produce a measurable signal, commonly:

  • GFP (Green Fluorescent Protein): Glows green under UV light
  • Luciferase: Produces bioluminescence
  • LacZ: Turns blue with X-gal

GFP in action: mice expressing Green Fluorescent Protein glow green under UV light, demonstrating gene expression

Image: Wikimedia Commons (Ingrid Moen et al., CC BY 2.5)

Example: The Toggle Switch

One of the first synthetic genetic circuits was the toggle switch (Gardner et al., 2000). It can exist in two stable states, like a light switch:

  • Two repressors that inhibit each other
  • Can be flipped between states with external signals
  • Maintains its state even after the signal is removed
üíª Software Analogy

The toggle switch is like a boolean flag with persistence:

// Pseudo-code for genetic toggle switch
let state = false; // Can be "on" or "off"

function inducer1Signal() {
    state = true;  // Flip to ON
}

function inducer2Signal() {
    state = false; // Flip to OFF
}

// State persists until next signal!

Applications of Genetic Circuits

  • Biosensors: Cells that detect environmental conditions (toxins, diseases)
  • Living therapeutics: Engineered bacteria that produce drugs only when needed
  • Biomanufacturing: Optimized production of chemicals and medicines
  • Cell-based computation: Performing logic inside cells

Connection to Longevity

Genetic circuits could enable:

  • Aging biosensors: Circuits that detect cellular aging markers
  • Smart senolytics: Systems that identify and eliminate senescent cells
  • Therapeutic release: Drugs produced only when and where needed
  • Microbiome engineering: Programmable bacteria for gut health
